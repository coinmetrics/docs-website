# Contracts Executed Successfully Cnt

## Definition

The sum count of contract calls successfully executed across all transactions in that interval. A contract call is the invocation of a contractâ€™s code by another contract or non-contract address. Only successful executions are taken into account.

| ----------------------------------- | ----------------- | ------------ | ----------- | ---- | ----- | -------- |
| Contracts Executed Successfully Cnt | TxContCallSuccCnt | Transactions | Internal    | Sum  | Calls | 1 day    |

## Details

* Contract creations and destructions are not included.

## Asset-Specific Details

* This metric is only available for ETH and ETC.
