# Flows Sent by Miners \(USD\)

## Definition

The sum of block rewards, fees, and other transfers received by all mining entities during an interval, excluding transfers from another mining entity. A mining entity is defined as an address that has been credited from a transaction debiting the 'FEES' or 'ISSUANCE' accounts in accordance with Coin Metricâ€™s Universal Blockchain Data Model \(UBDM\).
