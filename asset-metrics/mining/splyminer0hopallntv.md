# Miner Supply \(native units\)

## Definition

The sum of the balances of all mining entities. A mining entity is defined as an address that has been credited from a transaction debiting the 'FEES' or 'ISSUANCE' accounts in accordance with Coin Metricâ€™s Universal Blockchain Data Model \(UBDM\).
